# OmniSeller OS Two-Minute Demo Script

## 1. Boot

Open the app and show the desktop shell. Call out the seller stats: 18 orders today, Rs 24,500 revenue, and 3 low-stock SKUs.

## 2. Unified Orders

Run `show today's orders`.

Narration: "Every marketplace order lands in one inbox, with channel tags so the seller can process work from one queue."

## 3. Inventory Sync

Open Inventory and trigger the Amazon sale simulation.

Narration: "One warehouse, one truth. When Amazon sells one unit, the shared SKU count is reduced for Flipkart and Meesho too, preventing overselling."

## 4. GST Invoice

Run `generate invoice for order #4521`.

Narration: "The invoice app generates one clean GST format regardless of which marketplace produced the order."

## 5. Review Reply

Run `reply to this 1-star Meesho review`.

Narration: "The reply app drafts a polite, action-oriented response so the seller can recover the customer experience faster."

## Close

"The connectors are mocked for the demo, but the OS shape is real: shell, kernel, apps, connectors, and shared marketplace data."
